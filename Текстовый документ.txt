Собран production-ready scaffold Telegram-бота для изучения английского. Ниже — что уже есть и что нужно от вас.

Что создано

Prisma — 17 сущностей (все из ТЗ)

Docker Compose — PostgreSQL + приложение
Swagger — http://localhost:3000/docs
JWT Auth — внутренний REST API

Что нужно от вас
1. Обязательно
Что	Где взять	Куда
Telegram Bot Token
@BotFather → /newbot
.env → TELEGRAM_BOT_TOKEN
Gemini API Key
Google AI Studio
.env → GEMINI_API_KEY
JWT Secret
Любая длинная случайная строка
.env → JWT_SECRET
2. Системные зависимости
Node.js 20+
FFmpeg — для конвертации OGG → WAV
Windows: winget install ffmpeg или скачать с ffmpeg.org
Docker Desktop — для PostgreSQL
3. Запуск
cd d:\cursor_projects\english_speak_bot
copy .env.example .env
# Заполнить TELEGRAM_BOT_TOKEN и GEMINI_API_KEY
npm install
docker compose up postgres -d
npm run prisma:generate
npm run prisma:migrate:deploy
npm run db:seed
npm run start:dev
После запуска откройте бота в Telegram и отправьте /start.

Что ещё доработать (следующие этапы)
Scaffold готов к развитию, но для полного production понадобится:

STT/TTS — сейчас через Gemini; для production лучше Google Cloud Speech или Whisper
Webhook вместо long polling для Telegram
E2E-тесты сценариев бота

Reading / Writing — режимы заложены в schema, логика — в этапе 9 плана
Архитектура (кратко)
Telegram → Handlers → Domain Services → AiModule (Gemini)
                    → VoiceModule (FFmpeg + STT)
                    → Prisma → PostgreSQL
